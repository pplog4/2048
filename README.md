## msapp ファイルのインポート方法

1. このリポジトリから `.msapp` ファイルをダウンロードします。

2. [Power Apps ポータル](https://make.powerapps.com/)にアクセスし、必要に応じてログインします。

3. 任意のアプリを起動します。

4. 任意のアプリのStudio上部にある・・・→開くをクリックします。

5. 「参照」をクリックしてmsappファイルを選択します。
